# Omni Sprint

**Make sprint behave like a modern FPS:** sprint in any direction (forward, backward, strafe), keep toggle behavior intuitive, and preserve all original stamina/animation logic.

---

## Installation
Download the latest release and unpack the folder into the 7 Days to Die mods directory, just like any other mod.

## Server/Client

The mod is best played offline, if you want to use it on your server then you will have to disable EAC and your clients will need the mod also installed otherwise it will desync.
